1.0 Project Description.



HdrsSQLtoSH mimics AnySQLtoSH, the primary difference being that it allows
each .sql to have associated .hd1 and .hd2 files that will serve as the
shell surrounding the generated "batched" .sql query.



< filename >.hd1 header looks just like hdrAnySQL1.sh with mods specific to
setting up the SQL #procbodyhere section.
< filename >.hd2 header looks just likt hdrAnySQL2.sh with mods specific to
closing out the SQL #endprocbody section.
